# QR_Scan
auto QR scan.py
在运行脚本的时，可能会提示安装selenium，输入“pip install selenium” 运行即可。

同时运行脚本需要安装chromedriver.exe。 在https://chromedriver.chromium.org/下载对应chrome版本的chromedriver.exe ， 放置在C:\Program Files\Google\Chrome\Application即可。
